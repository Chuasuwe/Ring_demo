# 3D戒指展示网页

这是一个使用Three.js和WebGI技术实现的3D戒指展示网页。通过优雅的滚动交互，展示精美戒指的各个角度和细节。

## 特点

- 3D戒指模型的实时渲染
- 流畅的滚动交互体验
- 优雅的文字排版和动画效果
- 响应式设计，适配各种设备

## 技术栈

- Three.js - 3D渲染
- WebGI - 3D模型加载和处理
- TypeScript - 开发语言
- SCSS - 样式处理
- Parcel - 构建工具

## 本地运行

1. 克隆项目
```bash
git clone [你的仓库URL]
cd [项目目录]
```

2. 安装依赖
```bash
npm install
```

3. 启动开发服务器
```bash
npm start
```

4. 打开浏览器访问 `http://localhost:1234`

## 构建部署

构建生产版本：
```bash
npm run build
```
